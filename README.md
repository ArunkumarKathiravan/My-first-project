# ⏳ Smart Study Timer (Pomodoro CLI App)

A simple and effective **command-line study timer** built using Python.
This project helps students stay focused using the **Pomodoro technique** (25 min study + 5 min break).

---

## 🚀 Features

* ⏱️ 25-minute focus timer
* ☕ 5-minute break timer
* 🔁 Automatic session switching
* 🔔 Sound alert when time ends
* 📊 Session counter

---

## 🛠️ Tech Used

* Python (Core)
* Time module
* OS / Notification (optional)

---

## 📸 Demo

```bash
Start Study Session? (y/n): y
⏳ Study session started for 25 minutes...
✔ Time's up! Take a break.
☕ Break time: 5 minutes
```

---

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/smart-study-timer.git
```

2. Navigate to project folder

```bash
cd smart-study-timer
```

3. Run the program

```bash
python timer.py
```

---

## 💡 How It Works

* Starts a **study timer**
* After completion, switches to **break mode**
* Repeats cycle until user stops
